#Guide-API [![Build Status](http://tehnut.info/jenkins/buildStatus/icon?job=Guide-API)](http://tehnut.info/jenkins/job/Guide-API/)

Library mod for easy creation of guide books. Also allows book creation via JSON files.

##Issue Reporting

Please include the following:

* Minecraft version
* Guide-API version
* Forge version/build
* Versions of any mods potentially related to the issue
* Any relevant screenshots are greatly appreciated
* For crashes:
 * Steps to reproduce
 * Latest Forge log or crash log
